# **App Name**: Kabir's Showcase

## Core Features:

- Homepage Hero Section: Visually appealing hero section with a catchy tagline and brief introduction.
- Projects Portfolio: Display a grid of projects with descriptions and links to external resources or subdomains.
- Pulse Ecosystem Diagram: Interactive diagram showcasing the relationships within the Pulse application ecosystem.
- 3D Prints Gallery: Gallery to display 3D printed models with links to MakerWorld.
- Static Updates Feed: Chronological list of updates and changelogs.
- Contact Information: Section with links to social media and a contact form (email link).

## Style Guidelines:

- Primary color: Blue-purple (#4f46e5) to reflect innovation and creativity.
- Background color: Light gray (#f9fafb) to create a clean and modern feel.
- Accent color: Light purple (#6366f1) to highlight key elements.
- Font: 'Inter' sans-serif for a modern and readable text. Note: currently only Google Fonts are supported.
- Rounded corners (2xl) and card shadows for a modern, engaging look.
- Smooth hover transitions to enhance user interaction.
